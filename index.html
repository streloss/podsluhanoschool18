Правила форума

Соблюдение этих правил обязательно для всех пользователей форума.

  • 1. 18+ изображения полу-разрешен
    Можно загружать изображения для взрослых, но не должно быть видно открытых мест. За нарушение этого правила - бан по IP.
  • 2. Спамить сообщения запрещено
    Запрещено отправлять однотипные сообщения, флудить, создавать множество одинаковых постов. Наказание: бан по IP.
  • 3. Уважение к участникам
    Запрещены оскорбления, угрозы, унижение достоинства других пользователей.
  • 4. Конфиденциальность
    Не разглашайте личную информацию других пользователей без их согласия.
  • 5. Школьная тематика
    Форум создан для обсуждения школьных вопросов. По возможности придерживайтесь темы.

Администрация оставляет за собой право блокировать пользователей без предупреждения за нарушение правил.
